# Özür Sitesi 💕

Kız arkadaşın için özel hazırlanmış romantik özür sitesi.

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

Tarayıcıda http://localhost:3000 adresini aç.

## Vercel'e Deploy

1. GitHub'a push et
2. Vercel'e git (vercel.com)
3. "Import Project" ile GitHub repo'nu seç
4. Deploy et!

## Özelleştirme

- `app/page.tsx` dosyasındaki metinleri değiştirebilirsin
- Renkleri ve animasyonları istediğin gibi ayarlayabilirsin
- Fotoğraf eklemek istersen söyle!

Başarılar! 🍀
